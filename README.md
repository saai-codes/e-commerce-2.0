# 🛒 Saai Store – E-Commerce 2.0

## 📌 Project Overview

Saai Store – E-Commerce 2.0 is a modern and responsive frontend e-commerce website built using HTML, CSS, and JavaScript.
It allows users to browse products, add them to cart, and simulate an online shopping experience.

This project is an upgraded version (v2.0) with improved UI, better user experience, and additional features like cart and wishlist.

---

## 🚀 Features

* 🛍️ Product Listing Page
* 🔍 Search Functionality
* 🛒 Add to Cart System
* ❤️ Wishlist (Favorites)
* 📦 Checkout Page (Form + COD option)
* 📱 Fully Responsive Design
* ⚡ Smooth UI & Animations

---

## 🧠 Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout
* **JavaScript** – Functionality and interactivity
* **LocalStorage** – To store cart and wishlist data

---

## 💡 How It Works

* Users can browse products from the homepage
* Click on products to view details
* Add products to cart or wishlist
* Cart items are stored using localStorage
* Users can proceed to checkout and fill details
